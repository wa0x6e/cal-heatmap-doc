verticalOrientation

Whether the domains should be arranged on top of each other, or side by side.

verticalOrientation: boolean;

Default: false

Usage​

http://localhost:3000
Live Editor
const cal = new CalHeatmap();
// Set the options to true to view result
cal.paint({ verticalOrientation: false });
